DRUGRES

A tool for prediting drug resistance in cancer cell lines using machine learning approaches
Visualisation by: MDS6

DrugRes - Functionalities Guideline

  1. Upon launching the web application, users will encounter two primary buttons to choose from:
    • User Manual Prediction: This option is ideal for users who wish to upload their own dataset to predict drug sensitivity manually.
    • Default Dataset: Select this option if the user prefers to directly access drug sensitivity data from the default GDSC-CCLE drug-cancer dataset provided.
  2. Initial Assumption: Before selecting any of the two buttons, the web application will automatically assume that the default dataset option is selected. As a result, it will display the GDSC-CCLE drug-cancer dataset and the default output visualisation. Users can then proceed to choose between the two options based on your preference and requirements.
  3. Upon clicking the "User Manual Prediction" button, a dropbox will appear, allowing you to upload a single dataset. This dataset should consist of drug and cell line information in CSV format only. Since no prediction is made before user upload, the prediction result and output visualisation sections will remain blank. Please ensure that the uploaded dataset follows the specific format outlined in the "User Upload Data Format Requirement" provided below.
  4. Users can upload their dataset by either dragging and dropping the file into the dropbox or by clicking the "Browse" button to select the file manually. Once the dataset is uploaded, users need to click the "Make Prediction" button to initiate the prediction process. Please note that data processing may take some time, so we kindly ask for your patience. The web application will utilize the trained predictive model to make predictions based on the uploaded dataset. The prediction output will be presented in a tabular format, and an output visualisation will be provided to enhance the understanding of the prediction results.
  5. Users have the option to export either the prediction result or the default GDSC-CCLE dataset in CSV format. This functionality allows users to save the data for further analysis or reference.
  6. The prediction result table and the default GDSC-CCLE table both feature a sorting function. This enables users to rearrange the table's rows based on specific criteria, enhancing comprehension and ease of navigation.
  7. Filter buttons are available above the default dataset or the prediction table. Users can utilize these filters to manually search for specific drugs or cell lines of interest. The filtering options include manual key-in search and checkbox selection, providing flexibility in data exploration and analysis.

User Upload Data Format Requirement:

  1. The uploaded dataset should strictly adhere to the provided format. Below is a sample dataset for reference and users can download the sample dataset to serve as the template of the user upload data.
  2. The "isosmiles" column in the sample dataset can be obtained from the PubChem website https://pubchem.ncbi.nlm.nih.gov/ following these steps:
    • Click the “UPLOAD ID LIST” button.
    • Enter a list of drug IDs manually in the text field, separating each drug ID by a comma (e.g., 1013, 1014, 1015).
    • Alternatively, upload a CSV file containing a drug list, with one drug per line.
    • After entering or uploading the drug list, download the "isosmiles" in CSV format to retrieve the “isosmiles” of the drugs to be predicted. Document these”isosmiles” into the user upload dataset.

    Note:
    ** If the format of the uploaded dataset does not match the requirements, an error prompt message will notify the user that the uploaded dataset does not align with the specified format. You can upload the corrected dataset again.
    ** If there are any NA values in any rows of the user upload data, those rows will be removed upon making predictions.

Drug-Cell Sensitivity Table/ Prediction Output

The table provides information on drug-cell resistance prediction, with the following headers:

  • COSMIC_ID: Unique identifier for the cancer cell line.
  • CCLE_Name: The name of the cancer cell line.
  • DRUG_ID: Unique identifier for the drug.
  • DRUG_NAME: The name of the drug.
  • LN_IC50: The natural logarithm of the IC50 value, indicating the resistance of the drug to the cancer cell line.
  • Resistance_Cut-Off: The cut-off point of LN_IC50, distinguishing between high resistance and low resistance effects of the drug on the cell line.

Abbreviations

GDSC: Genomics of Drug Sensitivity in Cancer
CCLE: Cancer Cell Line Encyclopedia
Isosmiles: Isomeric Simplified Molecular Input Line Entry System

Explanation on LN_IC50 and Results Interpretation

  1. A lower LN_IC50 value indicates a more potent drug, reflecting higher sensitivity to the cancer cell line and lower resistance.
  2. The cutoff point of LN_IC50 is 3.7672284205249125. LN_IC50 values at or below 3.77 indicate drugs with high sensitivity to the cancer cell line (low resistance), while LN_IC50 values above 3.77 suggest low sensitivity to the cancer cell line (high resistance).
